Master Boot Record (MBR)

Master Boot Record, создаётся, когда вы создаете первый раздел на жестком диске, вероятно большинство важных дата структур записываются на него в этот момент. Это первый сектор на любом диске. Расположение track (cylinder) 0, side (head) 0, sector 1.

Master Boot Record содержит Partition Table на диске и маленький объем выполняемого кода. Master Boot Record находит стартовый системный раздел, расположенный на диске и загружает с него копию Partition Boot Sector в память. The Master Boot Record передает выполняемый код в Partition Boot Sector.
