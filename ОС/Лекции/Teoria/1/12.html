Основные команды MS-DOS

   Редактирование текстовых файлов.

Команды:

EDIT,FIND,TYPE

EDIT – Редактор MS-DOS Версия 2.0.026 Copyright (c) Microsoft Corp 1995.

EDIT [/B] [/H] [/R] [/S] [/<nnn>] [/?] [файл(ы)]

/B - Монохромный экран.

/H - Максимально возможное число строк.

/R - Загрузка файлов в режиме доступа только для чтения.

/S - Использование только коротких имен файлов.

/<nnn> - Загрузка двоичных файлов с разбиением на строки по <nnn> байт.

/? - Вывод справочной информации.

[файл] - Имя либо имена загружаемых файлов. Допускается использование

подстановочных знаков и нескольких образцов одновременно.

FIND - Поиск текстовой строки в одном или нескольких файлах.

FIND [/V] [/C] [/N] [/I] "строка" [[диск:][путь]имя_файла[ ...]]

/V Вывод всех строк, НЕ содержащих заданную строку.

/C Вывод только общего числа строк, содержащих заданную строку.

/N Снабжение отображаемых строк номерами.

/I Поиск без учета регистра символов.

"строка" Искомая текстовая строка.

[диск:][путь]имя_файла

Файл либо файлы, в которых производится поиск.

Если путь не задан, FIND производит поиск в тексте, введенном с клавиатуры,

либо переданном по конвейеру другой командой.

TYPE - Вывод на экран содержимого текстовых файлов.

TYPE [диск:][путь]имя_файла

Для получения краткой информации о соответствующей команде следует вызвать любую команду с параметром /?.
